## Overview

This folder contains datasets used across CAMIS examples for comparing methods across R, SAS, and Python.

## Datasets

| Dataset | Description | Purpose | Comments |
|------------------|-------------------|------------------|------------------|
| adcibc.csv | Clinical trial data with subject IDs, treatment dates, demographics, visits, and CIBIC scores | Used for longitudinal and clustered clinical trial analyses, including MMRM, GEE and Mixed effect models | Visit based outcome data(ADaM-style/ADQS) with subject and site level structure, treatment group and clinical endpoint values |
| adcibc.Rdata | Contains the adcibc clinical trial dataset stored as a single column RData object, with fields seperated bt commas | Used in R examples where the dataset is loaded with data('adcibc') | Requires parsing into multiple columns and formatting before analysis |
| apisrs.csv | Contains California school-level data with demographics, test scores, district information, and socioeconomic indicators | Used in regression, sampling, and multilevel modeling examples | Classic API dataset with many numeric predictors |
| blood_pressure.csv | Patient-level before/after blood pressure measurements | Used in paired t-tests and repeated-measures comparisons | Clean structure ideal for teaching/learning |
| blood_pressure_ties.csv | Same as blood_pressure but with integer BP values | Used in Wilcoxon signed-rank and nonparametric paired tests | The rounded datasets introduces more ties to illustrate behavior of non-parametric paired methods |
| bmt.sas7bdat | Time-to-Event clinical dataset with 137 observations across three treatment groups, includes survival time(T), a multi-state event indicator and pre-event waiting time | Used for Kaplan Meier estimation, Cox models, competing‑risk analyses, and ADaM‑style ADTTE derivations | This dataset includes two event types (1 and 2), enabling multi‑state and competing‑risk modelling |
| cached_mmrm_results.rda | A pre‑computed benchmarking results object containing convergence times, convergence rates, relative differences in parameter estimates, and missingness profiles across multiple mixed‑model implementations (mmrm, lmer, gls, glmmTMB, PROC GLIMMIX). Includes results for two endpoints (FEV and BCVA) under varying missingness conditions | To demonstrate model caching and reproducibility - Used to compare performance, numerical stability, and estimation accuracy across different MMRM and mixed‑model engines. Supports validation, reproducibility checks, simulation studies, and method comparison in teaching or documentation | Not a dataframe; contains fitted model objects - Contains six components: convergence time summaries (conv_time_fev, conv_time_bcva), relative difference tables (rel_diff_ests_tbl_fev, rel_diff_ests_tbl_bcva), convergence rates (conv_rate), and expected missingness patterns (df_missingness). Not patient‑level data; this is a benchmarking results bundle |
| colds.csv | Binary repeated‑measures dataset with subject IDs and indicators of cold occurrence at ages 12 and 14 (age12, age14) | Used for paired binary outcome analyses such as McNemar’s test, repeated‑measures logistic regression, and agreement/transition modelling | Demonstrates two‑time‑point binary outcomes with simple “Yes/No” responses; ideal for teaching paired categorical methods. |
| drug_trial.csv | Small three‑arm clinical trial dataset with treatment group, pre‑treatment score, post‑treatment score, and sex. Each row represents one subject with baseline and follow‑up measurements | Used for paired t‑tests, ANCOVA, linear models, and treatment‑effect demonstrations. Suitable for illustrating baseline adjustment, change‑score analysis, and subgroup comparisons | Demonstrates pre/post continuous outcomes across multiple treatment groups |
| htwt.csv | Anthropometric dataset with 237 observations containing subject row ID, sex, age in years, height in inches, and weight in pounds | Used for linear regression, correlation analysis, growth modelling, BMI calculation, and teaching relationships between age, height, weight, and sex | Clean continuous dataset ideal for demonstrating scatterplots, regression lines, sex‑stratified analyses, and height–weight modelling |
| jonck.csv | Dose–response dataset which contains response values across ordered dose groups | Used in trend analysis and ordered-group comparison methods | Demonstrate analysis of monotonic dose-response relationships, including non-parametric trend tests such as Jonckheere-Terpstra test |
| lung_cancer.csv | Lung cancer survival dataset with survival time, event/censoring status, demographics performance scores, nutrition/weight-loss variables, and treatment/dose indicators | Used in survival modeling and treatment comparision examples | Demonstrates time-to-event analysis with censoring, treatment groups, dose levels, baseline prognostic factors, and missing covariate values |
| Mall_Customers.csv | Contains customer demographic and purchasing related variables | Used for customer segmentation and clustering (k-means) examples | Demonstrate unsupervised learning methods such as k-means clustering using income and spending behaviour - Rename Genre variable to Gender |
| manova1.csv | Soil chemistry dataset with multiple mineral concentrations(multivariate) collected across different sites | Used in MANOVA and multivariate analysis | Demonstrates Comparision of multiple correlated response variables across environmental sampling locations. It contains al(aluminium), fe(iron), mg(magnesium), ca(calcium), na(sodium) |
| NCCTG_Lung_Cancer_Data_535_29.csv | NCCTG lung cancer survival dataset with patient index, institution/site, survival time, censoring/event status, age, sex, ECOG Performance, Karnofsky scores, meal calories, and weight loss | Used for survival analysis and prognostic modeling examples | Demonstrate time-to-event analysis with censoring, clinical baseline covariates, performance status measures, and institution-level variation |
| nhanes.csv | NHANES is survey data including sampling design variables | Used for complex survey analysis and weighted population inference examples | Demonstrate analysis of stratified, clustered, and weighted survey data using nationally representative health survey sampling design. It contains categorical and numeric variables i.e SDMVPSU-Primary Sampling Unit(Cluster), SDMVSTRA-Survey strata, WTMEC2YR-Sampling weight, HI_CHOL-high cholesterol indicator, RIAGENDR-Gender |
| polyps.csv | Contains polyp counts outcome with treatment and age | Used for count data regression and treatment effect examples | Demonstrates analysis of overdispersed count outcomes using methods such as Poisson and negative binomial regression |
| ps_data.csv | Contains participants demographic and body composition varaiables including Treatment group, sex, age, weight, and BMI category | Used for categorical data analysis and group comparison examples | Demonstrates analysis of treatment and demographic associations with BMI categories using contigency tables, logistic models and stratified comparisons |
| quarto_pkg_dependencies.csv | Contains Quarto file paths, R package dependencies, and package hashes/versions for CAMIS R based '.qmd' files | Used for reproducibility checks and project dependency tracking | Demonstrate documentation of package usage across analysis files, supporting reproducible rendering and environment management |
| resp_gee.xlsx | Contains repeated response data with response categories | Used in GEE modeling of longitudinal binary outcomes | Excel format; requires readxl |
| sas_disease.csv | Contain outcome measurements across combination of treatment groups and disease categories with missing values present | Used for factorial ANOVA and missing data handling examples | Demonstrate two-factor experimental analysis with treatment-by-disease comparisions, incomplete observations, and potential interaction effects |
| stat_method_tbl.csv | Table of statistical methods and metadata - Contains mappings between statistical methods and their corresponding SAS, R, Python, and comparison Quarto analysis files | Used for method navigation, documentation indexing, and cross-platform comparison workflows | Demonstrate structured organisation of statistical methodology resources across software platform and analysis categories |
| trial01.csv | Subject-level clinical trial dataset with binary outcome(aval) , treatment group, baseline covariate values, and indicator variables | Used for logistic regression, covariate adjustment, and treatment effect modelling examples | Demonstrate binary endpoint analysis with treatment, baseline adjustment, and categorical regional effects represented using dummy variables |
| whas500.sas7bdat | Worcester Survival and Cardiovascular follow-up data for 500 patients with demographic, clinical, cardiac history, and outcome variables | Used for survival analysis, cox proportional hazards modelling, cardiovascular risk assessment, and longitudinal follow-up analyses | Demonstrate time-to-event modelling using clinical predictors such as age,blood pressure, BMI, cardiac conditions, and mortality/follow-up indicators |
| WilcoxonSignedRank_TTR | Contains paired continuous measurements for the same subjects under two treatment conditions | Used for Wilcoxon signed-rank test, paired comparisons, and nonparametric within-subject treatment analyses | Demonstrates paired non-parametric testing for assessing median differences between two related treatment measurements when normality assumption may not hold |
